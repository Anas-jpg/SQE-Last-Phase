Modularity Requirements are

1.1 The System shall use central database for data sharing
1.2 The System shall use GPS Api for Maps sharing
1.3 The System shall divided into two main modules
  1.3.1 The System shall have customer modules
  1.3.2 The System shall have admin Modules
1.4 The Developer shall installed the Libraries to /usr/lib[64].
1.5  The Developer shall install the Headers to /usr/include.

1.6 The developer shall install the Executables to a location in the default system $PATH
