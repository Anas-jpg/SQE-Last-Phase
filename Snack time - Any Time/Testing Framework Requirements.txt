UI testing Requirements
1.1 The developer shall add flutter_test dependency
1.2 The developer shall create a test file with .dart extension
1.3 The developer shall create a class for test
1.4 The developer shall combine multiple tests in a group
1.5 The system shall use Intellij to run the Test

1.6 The system shall use Patrol framework for UI testing
	1.6.1 The System shall use CLI tool for orchestration of the test
