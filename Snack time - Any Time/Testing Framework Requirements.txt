UI testing Requirements
1.1 The test automation framework shall be be able to test every test case
1.2 The System shall generate the report of testing cases
1.3 The System shall be compatible with Testing Framework
1.4 The developer shall create a test file with .dart extension
1.5 The developer shall create a class for test
1.6 The developer shall combine multiple tests in a group
1.7 The system shall use IDE to run the Test

1.6 The system shall use Patrol framework for UI testing
	1.6.1 The System shall use CLI tool for orchestration of the test
